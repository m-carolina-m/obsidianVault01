Landscape tips:
1. Check sunset sunrise, sun position and weather in the location 
[http://www.photoephemeris.com/](http://www.photoephemeris.com/)
2. Use wide angle lens
10-20mm for APS-C
15-30mm for full frame
3. Pack light
1-2 lens
Tripod
Filters
Cleaning kit
Batteries
Memory card
4. Shoot in A set f/16 and ISO 100 for best quality. Use exposure compensation when necessary 
5. Maximize sharpness
Calculate correct doF (approx 1/3 distance). Set manual focus and take a shot to check by zooming in 
6. Use polarizing filter when necessary (too much light)
7. Use neutral density filter to slow shutter speed water and clouds blurring
8. Use neutral density graduated filter when there is a bright sky and a dark foreground to compensate and capture all the sky detail
9. Shot at the blue hour for cool colors
10. Shot on a grey day for drama effect
11. Get a good balance using composition (example: rule of thirds)
12. Include an interesting object in the foreground when possible 
13. Direct the viewers eye (lines)
14. Simplify try to use 1 or 3 objects 
15. At sunset make starburst. Sun just picking from an object f/16 or f/22
16. Look for symmetry
17. Shot ins the fog. Try to put the sun in front for better effect
18. Find different perspective and views. Shot up, or from the ground or looking down 

Panorama:
1. Manual mode 
2. Manual focus
3. Left to right
4. 20% overlapping or more in low light 
5. Manual white balance 
6. Vertical mode for more height
7. Be fast so you don't see the movement, especially when people are it the frame
8. Plan the amount on pictures, set the common points as reference 
9. Tripod recommended 
10. Switch off the screen. Take photos through viewfinder 
11. Try to be as parallel o the horizon as possible 
12. Take a marking photo to see where the pano starts and ends 
13. You can use bracketing to improve dark areas  (take several photos at the same position)
13. Auto stick them together using photoshop or Lightroom first the bracketed photos and then the panorama 

Interior building HDR vertoramas
1. Manual mode f/8 or the sharpest for the lens
2. Manual focus
3. Bottom to top
4. 30% overlapping or more in low light 
5. Manual white balance 
6. Landscape mode for more width
7. Be fast so you don't see the movement, especially when people are it the frame
8. Plan the amount on pictures, set the common points as reference 
9. Tripod recommended with nodal point adapter
10. Switch off the screen. Take photos through viewfinder 
11. Try to be as parallel to the vertical as possible 
12. Take a marking photo to see where the pano starts and ends 
13. Use bracketing to improve dark areas (take several photos at the same position, set A mode with the aperture selected, scan bottom to top and take note of the shutter speed range. Return to manual mode set aperture and middle shutter speed. set AEB so it goes below and above the shutter speed range, usually 3shots at +-2EV
13. Depending the ISO used you might first need to apply noise reduction to each photo 
14. Auto stick each section together using the bracketed photos using photomatix. Correct the tone mapping for each HDR and save them as 16-bit image 
15. Make the vertorama using photoshop photomerge. Select all the photos: use cylindrical, blend together,  vignette removal, geometric distortion correction 
16. Merge layers and correct the rotation 
17. Transform warp dragging the corner handles of the Warp box to the outside and the right and left edge handles to the inside
18. Crop the image
19. And post process as usual
